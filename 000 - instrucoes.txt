Passo 1: Configuração do Ambiente

 Instalar o Node.js ou verificar se ele foi instalado
node -v
npm -v

Criar a Pasta do Projeto

mkdir meus-testes-selenium
cd meus-testes-selenium

Iniciar o Projeto Node.js:
Dentro da pasta, execute o comando para criar um arquivo package.json. 
Este arquivo gerencia as dependências do seu projeto.

npm init -y

Instalar a Biblioteca Selenium WebDriver:
Agora, instale a biblioteca necessária:

npm install selenium-webdriver

Importante: A partir da versão 4.6, a biblioteca selenium-webdriver gerencia o 
download do driver do navegador (como o chromedriver) automaticamente. Você não 
precisa mais baixá-lo manualmente, o que facilita muito a vida!

Passo 2: Seu Primeiro Teste Automatizado (Hello, World!)

Vamos criar um teste simples que abre o Google, verifica o título da página e a fecha.

Crie um arquivo na sua pasta do projeto.

Volte ao terminal e execute o comando:

node primeiro_teste.js
